## 2024-05-18 - [GlobalNav Active Pill Accessibility & Polish]
**Learning:** Animated layout pills (e.g., Framer Motion active states in navigation bars) look visually pleasing but often lack semantic HTML accessibility attributes out of the box. Screen readers need `aria-current="page"` to understand the current active navigation link. Furthermore, inline styles for colors bypass CSS hover pseudo-classes, preventing proper inactive hover states which provide crucial interactive feedback to users. Keyboard users also need explicit `focus-visible` styling to navigate effectively.
**Action:** When implementing custom animated active navigation links, ensure you dynamically apply `aria-current="page"`, remove inline CSS colors in favor of class-based styling (like Tailwind) to allow for subtle `hover:bg` and `hover:text` on inactive items, and always add high-contrast `focus-visible` ring styles.
