# Nehal-Birthday

Vite + React app configured with:

- Three.js
- React Three Fiber + Drei
- Postprocessing (`@react-three/postprocessing`)
- Framer Motion
- GSAP
- Lenis
- Tailwind CSS (via `@tailwindcss/vite`)
- Google Fonts (Special Elite + Inter)
- Theatre.js (`@theatre/core` + `@theatre/studio` in dev)

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
